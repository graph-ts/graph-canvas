Drawing Circles

Two graph-canvas methods can be used to render circles on an HTML canvas element. The drawCircle() method renders a single circle and the drawCircles() method renders one or more circles.

Basic Circle

Use drawCircle() to render a single circle by providing a rendering context, position, radius, and style. 

{% include_relative draw-circles-0.js %}

Functions as Parameters

Functions can also be used as the radius and style parameters. These functions will be called with the position object as the only parameter and are expected to return a number for the radius and a Style object or array. In this example, we first create a node object that has a position and a random seed value between 0 and 10. Then we create two functions that, give a node object, return a circle radius and a style, respectively. The drawCircle() function then uses these functions to generate a radius and style when it renders the node.

Multiple Circles

The drawCircles() method can be used to render multiple circles that have the same radius and style. 

{% include_relative draw-circles-2.js %}

Multiple Circles With Functions as Parameters

Just like the drawCircle() method, drawCircles() can accept functions as the radius and style parameters. The drawCircles() method will perform better than a loop over drawCircle() if a static style is provided and not a function, otherwise it is simply a convenient way to render multiple circles.
